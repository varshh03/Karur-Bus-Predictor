# 🚌 Karur - puliyur Bus Arrival Prediction System

## 📌 Overview  
**Solving a problem I face daily** — uncertain bus timings during my college commute in Karur. Built ML model on self-observed real-world data to predict bus availability and crowd levels by time-of-day trends.

---

## 🚀 Features  
- 🕒 Predicts bus arrival times  
- 📊 Built using real-world, self-collected data  
- ⚡ Improves travel planning and reduces waiting time  
- 📍 Can be extended for real-time tracking  
- 🌐 Simple and user-friendly interface  

---

## 🛠️ Tech Stack  
- **Programming Language**: Python  
- **Machine Learning**: Random Forest (RFC)  
- **Libraries**: Pandas, NumPy, Scikit-learn  
- **Frontend (optional)**: Streamlit
